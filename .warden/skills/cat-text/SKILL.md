# Cat Text

Generates a short random text about cats and writes it to cat.md, overwriting whatever was there before.

## When to use me
- User says anything about writing cat text, "про котиков", or mentions cat.md
- User says "cat-text" or invokes this skill

## What I do
1. Generate a short (1-3 sentences) random paragraph about cats — funny, factual, or whimsical.
2. Write it to cat.md, overwriting the file.
3. Confirm it's done.

## Output
Confirmation message that cat.md was written. No emojis.
