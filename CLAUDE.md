# warden — инструкции для claude code

## проект

cli-агент по компу. python + textual tui + ollama. строгий минимализм.

## правила

- язык: русский, неформально, без смайлов
- не переписывай файлы целиком — только нужное
- перед большими изменениями спрашивай
- не добавляй зависимости без причины
- комментарии в коде на русском, коротко

## стек

- python 3.11+
- textual для tui
- ollama python sdk для llm
- pyautogui для мыши/клавы
- pillow для скринов
- playwright для браузера (не в mvp)

## стиль кода

- tabs, не spaces
- snake_case везде
- типизация обязательна (typing / dataclasses)
- без лишних абстракций — просто и понятно
- async где нужен стриминг

## структура

```
tui/       — всё про интерфейс
agent/     — логика агента, тулзы, промты
main.py    — точка входа, ничего лишнего
```

## визуал

никаких фоновых панелей и обводок. только текст + цвет + жирность.
цвета: cyan для варден, yellow для тулзов, red для ошибок, dim для метаинфо.
смотри theme.css за деталями.
